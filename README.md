# 인공지능 (Artificial-Intelligence)

## 딥 러닝(Deep Learning)


## 응용 영역

* 음성 인식, 스마트폰에서 가능한 이미지 자동 분류 스펨 메일 감지, 주식 시장, 의료 분야, 자동 주행 차량, 각종 로봇이나 드론 
* 이미지 분류와 같은 실용적인 컴퓨터 비전 예제
* 텍스트와 시계열 같은 시퀀스 데이터 처리 영역
